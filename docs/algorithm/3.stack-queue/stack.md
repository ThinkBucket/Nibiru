## 栈

栈是一种操作受限的线性表。只允许一端插入和删除数据，满足先进后出的特性。

## 如何实现一个栈

栈主要包括两个操作：

- 入栈`push()`：往栈顶添加一个数据
- 出栈`pop()`：从栈顶取一个数据
